Keep it simple

display: block

  • Takes up full width of parent
  • No elements side-by-side
  • Can have space above and below

...vs display: inline

  • Only takes as much space as it needs
  • Sits alongside other inline elements
  • Cannot have margins applied (hint: use display: inline-block;)
